# Assets

## Photo de profil

Place ta photo dans ce dossier avec le nom : `photo.jpg`

C'est la photo utilisée dans le Hero (polaroid).

Tu peux utiliser n'importe quel format : `.jpg`, `.jpeg`, `.png`, `.webp`
Si tu changes le nom ou le format, mets à jour la ligne d'import dans `src/sections/Hero.jsx` :

```js
import photo from '../assets/photo.jpg'
```
